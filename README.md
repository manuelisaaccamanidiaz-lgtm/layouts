# 🎨 Layouts Web – Colección de 55 Diseños con HTML & CSS

Este repositorio contiene una **colección de 55 layouts web** construidos únicamente con **HTML5 y CSS3**, enfocados en prácticas de maquetación, diseño visual y composición de estructuras web. Cada layout es una pieza independiente para experimentar o aprender cómo se construyen diferentes estructuras y patrones visuales sin JavaScript. :contentReference[oaicite:1]{index=1}

#🚀 ¿Cómo Ejecutarlo Localmente?
Sigue estos pasos para clonar el proyecto y ver los layouts en tu equipo:

# Clona el repositorio
git clone https://github.com/manuelisaaccamanidiaz-lgtm/layouts.git

# Entra al proyecto
cd layouts

# Abre un archivo HTML en tu navegador
open html/layout1.html

#🤝 Contribución

Este proyecto está abierto a contribuciones. Si quieres mejorar estilos, agregar nuevos layouts o corregir detalles:

> - Haz un fork del repositorio.
> - Crea una rama con tu feature: git checkout -b feature/nuevo-layout.
> - Haz commit: git commit -m "Agrega nuevo layout".
> - Envía un pull request describiendo tu aporte.
